


자바 스크립트 개발도구

1. Git : 협력, 버전 컨트롤 도구
2. Node : 브라우저 밖에서 자바스크립트를 실행할 수 있게 하는 도구, 노드와 함께 설치되는 npm은 리스트의 다른 도구를 설치할 때 필요함.
3. Gulp : 반복적인 개발 작업을 자동화하는 빌드 도구/ Grunt도 있음.
4. Babel : ES6 코드를 ES5코드로 변환하는 트랜스 컴파일러
5. ES Lint : 자주 하는 실수를 피하고 더 나은 프로그래머가 되도록 돕는 린트 프로그램



npm은 설치된 패키지 관리
패키지 = 애플리케이션 또는 코드 샘플, 프로젝트에서 사용할 모듈 또는 라이브러리
